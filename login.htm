This is an online key generation service for registered users. 

To obtain a license key, you need to indicate:
  • the number of the DOSPRN order and
  • the email address specified when purchasing the license.
You also need to specify a unique registration code:
  • open the main DOSPRN window
  • click on the 'Registration' button and copy data from the 'Code' field

You need to specify the previous license key if you want to get a replacement instead of a previously issued one.










If you're having trouble you can contact me via email.
