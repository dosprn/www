  • COMPATIBILITIES

    • How to use DOSPRN for printing reports from DOSBox Mega Build 6 (DOSBox MB6)?

      Here you can download DOSBox Mega Build 6 release with bugfixes.

      DOSPRN automatically sets the required DOSBOX MB6 configuration settings if you enabled the "Enable integration DOSBOX, vDos, vDosPlus" option during installation. You just need to start DOSBOX MB6, run the DOS program and print the reports.

      Otherwise you need to set up the DOSBOX MB6 configuration manually.
      Chose "All Programs-DOSBox MB6-Edit Configuration" option from the main menu and modify "parallel" section in the such way:

      ............
      [parallel]
      parallel1=file dev:lpt1
      parallel2=disabled
      parallel3=disabled
      ............
