Command line options

Now you can run DOSPRN from the batch files and console window with some command line options. You can combine command line options one between another.

The list of command line options supported by DOSPRN is here:

Start DOSPRN with a separate profile:
"C:\Program Files\DOSPRN\DOSprn.exe" /profile:ProfileName

Start DOSPRN for printing specified file silently:
"C:\Program Files\DOSPRN\DOSprn.exe" D:\full_path\filename.ext

Start DOSPRN for printing specified file with a small printers, fonts and margins selection dialog:
"C:\Program Files\DOSPRN\DOSprn.exe" -dialog D:\full_path\filename.ext

Start DOSPRN for printing on specified printer:
"C:\Program Files\DOSPRN\DOSprn.exe" /printer:"Canon iP4200"

Start DOSPRN with enabled/disabled 'Capture port' option:
"C:\Program Files\DOSPRN\DOSprn.exe" /capture:off

Start DOSPRN for monitoring separate spool file (with /capture=off only):
"C:\Program Files\DOSPRN\DOSprn.exe" /spool:"D:\SpoolDir\SpoolFile.ext"

Start DOSPRN in the enabled/disabled 'Autoprint' mode:
"C:\Program Files\DOSPRN\DOSprn.exe" /autoprint:off

Start DOSPRN for printing with a specific title:
"C:\Program Files\DOSPRN\DOSprn.exe" /title:"Printing Title"

Start DOSPRN for printing into the specified output file:
"C:\Program Files\DOSPRN\DOSprn.exe" /output:"C:\TEMP\REPORT.PDF"

Start DOSPRN for printing on specified paper:
"C:\Program Files\DOSPRN\DOSprn.exe" /paper:"Letter"

Start DOSPRN for printing in specified orientation mode:
"C:\Program Files\DOSPRN\DOSprn.exe" /orientation:"Landscape"

Start DOSPRN for printing with specified font name:
"C:\Program Files\DOSPRN\DOSprn.exe" /font:"Courier New"

Start DOSPRN for printing with specified default font size:
"C:\Program Files\DOSPRN\DOSprn.exe" /size:"11"

Start DOSPRN for printing with specified default CPI (characters per inch density):
"C:\Program Files\DOSPRN\DOSprn.exe" /cpi:"12"

Start DOSPRN for printing with specified default LPI (lines per inch density):
"C:\Program Files\DOSPRN\DOSprn.exe" /lpi:"6"

Start DOSPRN for printing in the specified escape sequences emulation mode:
"C:\Program Files\DOSPRN\DOSprn.exe" /sequences:"IBM proprinter"

Start DOSPRN for printing with specified default margins (top;left;right;bottom):
"C:\Program Files\DOSPRN\DOSprn.exe" /margins:"10;20;10;0"

Start DOSPRN for printing in the specified code page emulation mode:
"C:\Program Files\DOSPRN\DOSprn.exe" /codepage:"737"

Start DOSPRN with a 'print preview' mode on or off:
"C:\Program Files\DOSPRN\DOSprn.exe" /preview:on
"C:\Program Files\DOSPRN\DOSprn.exe" /preview:off

Start DOSPRN with a 'saving into PDF' mode on, off or into the specified file:
"C:\Program Files\DOSPRN\DOSprn.exe" /pdf:on
"C:\Program Files\DOSPRN\DOSprn.exe" /pdf:off
"C:\Program Files\DOSPRN\DOSprn.exe" /pdf:D:\full_path\filename.pdf

Start DOSPRN with a 'saving into PDF without dialog' mode on/off:
"C:\Program Files\DOSPRN\DOSprn.exe" /autosavepdf:on
"C:\Program Files\DOSPRN\DOSprn.exe" /autosavepdf:off

Start DOSPRN with an 'open PDF after saving' mode on/off:
"C:\Program Files\DOSPRN\DOSprn.exe" /openpdf:on
"C:\Program Files\DOSPRN\DOSprn.exe" /openpdf:off

Start DOSPRN with a specified 'folder to save PDF':
"C:\Program Files\DOSPRN\DOSprn.exe" /pdffolder:"D:\full_path_to_save_pdf\"

Start DOSPRN with a specified 'filename pattern to save PDF':
"C:\Program Files\DOSPRN\DOSprn.exe" /pdftemplate:"(dosprn)_yyyy_mm_dd_hh_nn_ss(.pdf)"

Start DOSPRN in the non-resident mode. It will be unloaded from the memory on 'Close' command:
"C:\Program Files\DOSPRN\DOSprn.exe" /noresident

Unload DOSPRN from the memory beforehand with command:
"C:\Program Files\DOSPRN\DOSprn.exe" /exit
